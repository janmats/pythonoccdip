Руководство пользователя


Для того, чтобы построить фигуру, задайте ее описание в поле "Введите запрос".
        Описание должно четко соответствовать цели и включать только названия фигур и размеры (в формате вещественного числа). Например:
        - Куб со стороной 10.0
        - Цилиндр с радиусом 5.0 и высотой 12.0
        - Куб со стороной 10.0 минус куб со стороной 5.0
        Для построения фигуры нажмите на кнопку "Построить". Будет отображена страница с трехмерным изображением фигуры. Изображением можно управлять: вращать, приближать, удалять

Для просмотра сгенерированного ChatGPT кода фигуры на странице с иозображением фигуры нажмите на кнопку "Code". Код фигуры будет отображен в раскрывшемся поле внизу.

В случае, если приложение не может построить фигуру по сгенерированному ChatGPT коду (например, если код содержит ошибки), оно построит последнюю успешную фигуру.
        Посмотреть код фигуры, которую не удалось построить, можно нажав на кнопку "Code". Посмотреть полученную ошибку можно, нажав на кнопку "Error"
